静态属性 ​

这里的 静态属性 是指挂载在 构造函数 的 一级属性，非常少使用

instances ​

返回全部播放器实例的数组，假如你想同时管理多个播放器的时候，可以用到该属性

▶ Run Code
js
console.info([...Artplayer.instances]);

var art = new Artplayer({
    container: '.artplayer-app',
    url: '/assets/sample/video.mp4',
});

console.info([...Artplayer.instances]);
