全局属性 ​

这里的 全局属性 也是指挂载在 构造函数 的 一级属性，属性名字全部都是大写的形式，未来容易发生变动，基本上用不到

DEBUG ​

是否开始 debug 模式，可以打印出视频全部的内置事件，默认关闭

▶ Run Code
js
Artplayer.DEBUG = true;

var art = new Artplayer({
    container: '.artplayer-app',
    url: '/assets/sample/video.mp4',
});
